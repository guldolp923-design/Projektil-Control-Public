# Projektil Control Public

Dieses Repository ist die oeffentliche Release-Quelle fuer **Projektil Control**.

## Zweck
- Bereitstellung von Windows-Installern (`.exe`)
- Bereitstellung von Release-Informationen fuer die integrierte Update-Pruefung
- Transparente Release-Historie ohne Offenlegung des privaten Quellcodes

## Aktuelle Version pruefen
- Oeffne den Bereich **Einstellungen -> Update** in der App.
- Klicke auf **Auf Updates pruefen**.
- Bei neuer Version erscheint **Download & Update**.

## Download
Die aktuellen Installationsdateien findest du unter [Releases](../../releases).

## Hinweise
- Dieses Repository enthaelt nicht den vollstaendigen privaten Entwicklungsstand.
- Support und Fehlerberichte bitte ueber die angegebenen Kontaktwege.
