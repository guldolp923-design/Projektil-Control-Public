# Security Policy

## Supported Versions
Aktiv unterstuetzt wird die aktuellste Beta- oder Stable-Version in diesem Repository.

## Meldung von Sicherheitsproblemen
Bitte keine Sicherheitsluecken oeffentlich in Issues posten.

Stattdessen:
- Kontaktaufnahme per privater Nachricht an den Maintainer
- Oder E-Mail mit reproduzierbaren Schritten und Impact-Beschreibung

Wir bestaetigen den Eingang schnellstmoeglich und priorisieren reproduzierbare Befunde.
